# 🍕 Pizza Sales Analysis using SQL

This project analyzes a pizza restaurant's sales data using SQL to uncover insights about order patterns, revenue, and customer preferences.

---

## 📌 Project Objectives

- Analyze pizza sales performance
- Identify popular pizza types, sizes, and categories
- Track customer ordering patterns
- Understand revenue distribution across products

---

## 🛠 Tools Used

- SQL (MySQL or PostgreSQL)
- DBeaver / DB Browser (for SQL execution)
- Excel (optional, for pre-analysis)

---

## 📊 Key Insights

- Total Orders Placed
- Total Revenue Generated
- Highest Priced Pizza
- Most Common Pizza Size
- Top 5 Most Ordered Pizza Types
- Category-Wise Quantity Ordered
- Order Distribution by Hour
- Average Number of Pizzas Ordered Per Day
- Top 3 Pizza Types by Revenue
- Percentage Revenue Contribution by Pizza Type
